\doxysubsection{Detailed Description}
Base class providing the main application loop and shared resources. 

\doxylink{class_c_base4618}{CBase4618} defines a standard run loop that repeatedly calls update and draw. The loop exits when the user presses the \textquotesingle{}q\textquotesingle{} key. Derived classes implement application-\/specific behavior.

Handles hardware input/\+output and communication with the embedded system. 

This method is called once per iteration of the main run loop and is intended to perform all low-\/level GPIO, ADC, and communication-\/related operations. The base class does not implement this method; derived classes must define the specific hardware interactions required by the application.

Typical responsibilities include:\+
\begin{DoxyItemize}
\item Reading analog inputs (e.\+g. joystick positions)
\item Reading debounced digital inputs (e.\+g. buttons)
\item Sending output commands (e.\+g. LEDs)
\end{DoxyItemize}

Application-\/level logic and rendering must not be performed in this method.
