Detailed Description

CControl class for communicating with the TM4C123G using the ELEX4618 serial protocol.

Protocol format (ASCII): GET: "G <type> <channel>\n" SET: "S <type> <channel> <value>\n"

Expected reply (ASCII): "A <type> <channel> <value>\n"

type: 0 = DIGITAL 1 = ANALOG 2 = SERVO
