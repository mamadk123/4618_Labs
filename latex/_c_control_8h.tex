\doxysubsection{Detailed Description}
\doxylink{class_c_control}{CControl} class for communicating with the TM4\+C123G using the ELEX4618 serial protocol. 

Protocol format:\+ GET:\+ "{}\+G $<$type$>$ $<$channel$>$\textbackslash{}n"{} SET:\+ "{}\+S $<$type$>$ $<$channel$>$ $<$value$>$\textbackslash{}n"{}

type:\+ 0 = DIGITAL 1 = ANALOG 2 = SERVO
